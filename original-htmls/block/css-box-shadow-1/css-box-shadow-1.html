Box shadow
Shadows can be added to a box using -ah-box-shadow. For -ah-box-shadow: 10px 10px 10px 10px darkgray, the values, in order, are: horizontal offset of the shadow; vertical offset of the shadow; blur radius; shadow expansion; shadow color. By adding inset to the value, an inner shadow can be created.

-ah-box-shadow: 10px 10px 0px 0px darkgray;

-ah-box-shadow: 10px 10px 5px 0px darkgray;

-ah-box-shadow: 10px 10px 0px 8px darkgray;

-ah-box-shadow: inset 10px 10px 0px 0px darkgray;

-ah-box-shadow: -10px -10px darkgray;

-ah-box-shadow: 10px 10px dimgray, 20px 20px darkgray, 30px 30px lightgray;
