Keep adjacent blocks on the same page
Ordinarily, a page break occurs only when the text extends to the bottom of the page. However, it is possible to impose keep conditions relative to different blocks and place them in the same page. There placed Q3 in the end of the second page and A relative to Q3 is placed at the top of the third page. This is an ordinary placement.
