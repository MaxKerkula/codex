Double-wave border form
In Antenna House Formatter V7.3 and later, -ah-border-wave-form: can also be used to specify the double-wave form. The third attribute value is valid for border-style: double-wave; and indicates the wave width. If the value is omitted or is auto, it is assumed to be 0.5.
border-style: double-wave;
border-style: double-wave; -ah-border-wave-form: 6 0.125 0.5;
border-style: double-wave; -ah-border-wave-form: 2;
border-style: double-wave; -ah-border-wave-form: 5;
border-style: double-wave; -ah-border-wave-form: 10;
border-style: double-wave; -ah-border-wave-form: 10 0.7;
border-style: double-wave; -ah-border-wave-form: 10 0.3;
border-style: double-wave; -ah-border-wave-form: 10 0.3 2;
