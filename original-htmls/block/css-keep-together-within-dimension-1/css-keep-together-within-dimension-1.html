Limiting the height to which 'keep-together' applies
When ‘keep-together.within-page’ or ‘keep-together.within-column’ is not ‘auto’, specify ‘axf:keep-together-within-dimension’ to limit the height for which the ‘keep-together’ applies. See “axf:keep-together-within-dimension” in the Online Manual. ‘axf:keep-together-within-dimension’ is available in AH Formatter V6.5 and later.
For example, if you want to specify keep-together.within-page="always" for a block but you also want the block to break if it exceeds the height of the page, then specify it as follows: <fo:block keep-together.within-page="always" axf:keep-together-within-dimension="100vh">

-ah-keep-together-within-dimension: all;
There is no limit on the height of the block (shown with red border): even if the block exceeds the column height, it is output as is.
