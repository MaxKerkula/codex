Barcode
This is an example of embedding barcodes directly in the document as images by using Antenna House Formatter Barcode Generator Option. A barcode font isn't used and a generated image of a barcode (PNG or SVG) is embedded. Data Matrix and PDF417 formats are available from AH Formatter V6.3 R1. Aztec Code formats are available from AH Formatter V7.1 R1. "Antenna House Formatter Barcode Generator Option" must be purchased separately in order to generate barcodes.
