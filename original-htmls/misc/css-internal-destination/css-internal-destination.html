Internal-destination
The internal-destination property of the basic-link element is used to create internal links in a PDF document. Antenna House has extended the value of this property to allow linking to specific pages and changing the PDF view. The following link goes to page 3 of this document and zooms the document out to 50%.
link to page 3 and zoom 50%
-ah-internal-destination: 3#zoom=50;
