Format
The format property is used to convert formatter-generated numbers like page numbers to different counting methods. Antenna House Formatter has extended this property to include many numbering systems and number formats. The format property is applied to a page sequence and this document is set to start on page 13 for demonstration purposes.
format="1" (default) page number
format="௧" (Tamil) page number
format="I" (Roman) page number
format="Ⓐ" (Circled Latin Capital) page number
