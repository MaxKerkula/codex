Font variant extension
Formatter has extended the values of the font-variant property to allow for many different values. Examples of individual values follow. Please see the Formatter documentation for information about which values are valid to combine together. Note that not all fonts can have all of the variant properties applied to them.
font-variant="normal" The Quick Brown Fox Jumps Over The Lazy Dog
font-variant="all-small-caps" The Quick Brown Fox Jumps Over The Lazy Dog
font-variant="normal" The Quick Brown Fox Jumps Over The Lazy Dog
font-variant="normal" The Quick Brown Fox Jumps Over The Lazy Dog
