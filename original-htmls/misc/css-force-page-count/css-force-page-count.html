Force page count
Formatter has extended the values of the force-page-count property to allow for customization of the amount of pages in a document or in a page sequence. The value doubly-even forces ending on multiples of 4. A second extended value forces a document or page sequence to end on specific multiple of a number plus another number of blank pages. This page sequence has the value force-page-count="end-on 6 3". There are 5 pages on content so the total document size will be 9 page(1 set of 6 pages and 3 following).
