Overflow
The overflow property is used determine the behavior of Formatter when the contents of an area format to a size greater than the area's defined boundaries. Antenna House has extended the value of this property to the following values overflow: replace; allows for replacement of overflowing contents, similar to how spreadsheets replace number too large to display in cells with "*****". The string to replace the overflow with is defined as the value of the extension element -ah-overflow-replace. overflow: condense; allows the text to be condensed according to the value of the -ah-overflow-condense extension property. These extended values only apply to block-container and inline-container elements.
