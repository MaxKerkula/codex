Page numbers with mixed horizontal and vertical pages
By specifying style @page-rule, page number are counted backwards, in descending order. Together with -ah-reverse-page: true, this makes it possible to generate PDF output with sequential page numbers on horizontal writing-mode pages in a vertical writing-mode document. In this sample, ‘-ah-reverse-page’ and ‘-ah-reverse-page-number’ are specified for the index on pages 4 and 5. ‘-ah-reverse-page-number’ is available in AH Formatter V6.3 and later.
