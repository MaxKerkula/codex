Omitting table header/footer at a column break
The XSL Specification includes table-omit-header-at-break and table-omit-footer-at-break properties for omitting the table header or table footer at a page break. However, because these properties only apply at a page break, the table header and table footer are generated when a column break occurs in a table on a multi-column region. When the extension property value 'column' is specified for table-omit-header-at-break and table-omit-footer-at-break, Antenna House Formatter omits the table header and footer when splitting the table across columns but does not omit the header and footer when splitting across pages.
