table-row widows/orphans
Specifies the number of table rows that should remain at the bottom (column) of the page or at the top (column) of the page. Use ‘-ah-table-row-widows’ for the number of table-rows at the top of the page, and use ‘-ah-table-row-orphans' for the number of table-rows at the bottom of the page. In each case, specify an integer value for the <tbody> element. This feature is available in AH Formatter V7.0 and later.
