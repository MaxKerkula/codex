Page numbers
To add page numbers in CSS, utilize the @page rule combined with pseudo-elements. Define the style and position, then use the counter(page) function within the content property. This will automatically generate and display the current page number in the specified position and style on each page. This example uses the following CSS code: content: "- " counter(page) " -"
