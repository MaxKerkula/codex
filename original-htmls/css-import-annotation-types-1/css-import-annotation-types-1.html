Keeping the annotation in the embedded PDF

When Embedding a PDF in a PDF, annotations existing in the embedded PDF can be retained by specifying import-annotation-types in the Option Setting file or PDF settings in Antenna House Formatter. Also it can be specified in Other page in the PDF Option Setting dialog in GUI. You can specify the type of annotation as "Text", "Link" individually or specify "All" to embed all embeddable annotations. Annotations that are not specified will be ignored.
