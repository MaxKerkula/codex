Display the alternate text of an image
Using ‘-ah-display-alttext’, specifies whether to display the alternate text of an image. This feature is available in AH Formatter V7.0 or later.
The following specifications are commonly used in this sample.
-ah-alttext: 'this content shows up if the parent image is missing'
src="img/button-none.png" is specified in <img> tag, but the image is missing. The alternate text is displayed instead.
-ah-display-alttext: true: The alternate text is displayed, the image is not displayed.
-ah-display-alttext: false: The alternate text is not displayed.
-ah-display-alttext: auto: The value specified by display-alttext in the Option Setting File is adopted.
