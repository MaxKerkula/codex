Column rule
You can draw a ruled line between columns by specifying -ah-column-rule-*. The color of the column rule can be specified by -ah-column-rule-color, the length can be specified by -ah-column-rule-length, the style can be specified by -ah-column-rule-style, and the width can be specified by -ah-column-rule-width. In AH Formatter V7.2 or later, a shorthand that sets border-width, border-style, color, and column-rule-display using -ah-column-rule is effective. On the first page, -ah-column-rule-style: solid, -ah-column-rule-width: thick, -ah-column-rule-color: #BBFFBB; are specified to the column rule.
