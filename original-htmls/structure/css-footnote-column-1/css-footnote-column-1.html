Column-width footnotes
In a multi-column document, you can include footnotes to provide additional information or references. By using the column property nested within a user-defined footnote class, you have control over how footnotes are displayed. When you insert a footnote at the bottom of a page, you can specify whether it should be displayed below its respective column or span across all columns using column-span: all.
