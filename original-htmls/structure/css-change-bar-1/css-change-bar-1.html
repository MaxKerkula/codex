Change Bar
Here is an example of displaying a change bar. By the use of <span> you can begin a change-bar with inline CSS properties. You can specify the position of a change bar object on the left/right, inside/outside of the block by ‘change-bar-placement’. ‘change-bar-color’, ‘change-bar-offset’, ‘change-bar-style’, and ‘change-bar-width’ are properties of <span> that specify the color, offset from the body, line style, and line width of the change bar, respectively.
