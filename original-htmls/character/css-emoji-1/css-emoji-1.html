CSS Emoji Support
Antenna House Formatter V7.3 supports the emoji (UTS#51: Unicode Emoji) is partially supported. Please refer to the online manual "Emoji" for details.

Support for vertically written emoji

Emoji characters in veritcal writing blocks are correctly displayed by specifying -ah-text-combine-horizontal: emoji;
