Glyph Orientation
Glyph Orientation can be set. ‘glyph-orientation-horizontal’ and ‘glyph-orientation-vertical’ when ‘writing-mode’ is ‘tb-rl’ can be specified for horizontal text.
    Example:
  1. glyph-orientation-horizontal: 0deg;
  2. glyph-orientation-horizontal: 90deg;
  3. glyph-orientation-horizontal: 180deg;
  4. glyph-orientation-horizontal: 270deg;
