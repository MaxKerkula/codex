Intruding the inline element into the punctuation
By specifying ‘axf:intrude-into-punctuation’, the inline element is intruded into the punctuation mark. The setting works when it is Japanese and '-ah-punctuation-trim: adjacent;' is specified and the available values are ‘none’ (default value), ‘start’ and ‘end’. ‘none’ does nothing. ‘start’ intrudes the inline elemenet into the punctuation mark when the character immediately before the start side is a full-width close parenthesis or a full-width punctuation mark, and ‘end’ does so when the character immediately after the end side is a full-width open parenthesis by handling each character as half-width. See ‘-ah-punctuation-trim’ for punctuation marks that you want to specify. This feature is available from AH Formatter V7.
