Hyphenation of hyphenated words
You can use -ah-hyphenate-hyphenated-word to control whether or not a hyphenated word (for example, “hyphenated-word” in these samples) can be further hyphenated.
When -ah-hyphenate-hyphenated-word: true; (default value) is specified, normal hyphenation is preformed when the existing hyphen is not at the end of line.
a padding padding padding padding padding padding padding hyphenated-word
padding padding padding padding padding padding padding padding hyphenated-word
When -ah-hyphenate-hyphenated-word: false; is specified, an already hyphenated word will not be further hyphenated. A line break can occur only at an existing hyphen.
a padding padding padding padding padding padding padding hyphenated-word
padding padding padding padding padding padding padding padding hyphenated-word
