Line continued mark
A line continued mark can be added by using ‘-ah-line-continued-mark'. A line continued mark can be added with block-level element as a unit. "-" (hyphenation) is specified in this case. Also, you may specify a line continued mark’s color by ‘-ah-line-continued-mark-color’, ‘-ah-line-continued-mark-font-family’, etc., and the background color of a new paragraph by ‘-ah-line-continued-mark-background-color’.
