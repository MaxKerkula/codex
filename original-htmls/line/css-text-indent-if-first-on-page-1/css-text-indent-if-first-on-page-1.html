‘text-indent’ of a block at the top of the page
Specify ‘-ah-text-indent-if-first-on-page’ for the ‘text-indent’ of a block at the top of the page or the top of the column. When the block comes to the top of the page or column, if the first line is on that page or column, the value of this property substitutes for the value of ‘text-indent’. This feature is available in AH Formatter V7.0 or later.
