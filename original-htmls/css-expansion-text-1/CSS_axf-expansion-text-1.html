Specifying expansion text for tags in Tagged PDF
When you create a Tagged PDF for accessibility, you can use "-ah-expansion-text" to specify expansion text that specifies descriptive text for abbreviations, etc. in the Tagged PDF. The setting of ‘axf:expansion-text’ is available from AH Formatter V7.2.
In this sample, the expansion text is specified by -ah-expansion-text as shown below. When the text of a PDF document as a Tagged PDF output from Antenna House Formatter is read aloud by screen reading software, the expansion text is also read aloud along with the abbreviation.
<div> <span style = "-ah-expansion-text: 'Cascading Style Sheets';">CSS</span> is one of the specifications that indicates how to qualify HTML and XML elements. </div>
