Character outline
-ah-text-stroke: specifies the outline stroke of a character, whereas color sets its fill color. Set it as -ah-text-stroke: 2pt green, and the width and color of the character stroke will be set accordingly. For more details on -ah-text-stroke, please refer to the Online Manual. This feature is available in AH Formatter V6.5 and later.

-ah-text-stroke: 2pt green; color: white;
