The height of the dropped initial is set to the cap-height or x-height of the first line by specifying ' -ah-initial-letters-first-line-head-height'. However, some fonts do not have the cap-height and x-height information, so the dropped initial characters and the height of the first line will not match. In such a case, it can be adjusted by specifying in the Font Configuration File of Antenna House Formatter. For example, the setting in the font construction file is as follows: 

<font-folder path="C:\*****">
  <font-entry file="frank.ttf" cap-height="0.46"/>
</font-folder>
