The CSS properties -ah-table-omit-header-at-break and -ah-table-omit-footer-at-break are used to prevent a table header or footer from appearing after a table breaks across a page but allow the header or footer to appear when the table breaks across a column.

These CSS properties have been extended to allow for a value of column. This value will omit the header or footer when the table breaks across a column but not when it breaks across a page.
