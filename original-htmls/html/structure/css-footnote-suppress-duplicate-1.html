Suppressing duplicate footnotes within the same page

When the same footnote is allotted at several positions within a same page, using -ah-suppress-duplicate-footnote prevents them from being duplicated and displayed within the same footnote region. In this page, a duplicated display is controlled with -ah-suppress-duplicate-footnote: true;
