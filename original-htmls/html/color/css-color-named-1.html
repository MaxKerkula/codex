Named colors

XSL 1.1 defines 16 color keywords: ‘aqua’, ‘black’, ‘blue’, ‘fuchsia’, ‘gray’, ‘green’, ‘lime’, ‘maroon’, ‘navy’, ‘olive’, ‘purple’, ‘red’, ‘silver’, ‘teal’, ‘white’, and ‘yellow’. Antenna House Formatter also supports the 131 extended color names from CSS Color Module Level 3 (and SVG 1.0) plus ‘rebeccapurple’ that is defined in CSS Color Module Level 4.
