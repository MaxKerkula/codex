Text aligment in a table cell

Use the XSL 1.1 -ah-display-align and vertical-align property to align the blocks of text in a table cell to the top, bottom, or center of the cell. Antenna House Formater has an extension: justify value that adjusts the space between lines so that the text fills the table cell.

vertical-align: top; vertical-align: bottom; -ah-display-align: center; -ah-display-align: justify;
