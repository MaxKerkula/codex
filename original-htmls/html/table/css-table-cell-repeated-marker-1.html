Content of a table cell split by a page break

The content of a cell that is being split by a page break can be specified using -ah-table-cell-repeated-marker. When -ah-repeat-cell-content-at-break: true is specified to a cell and -ah-table-cell-repeated-marker is also included in the cell, the content specified by -ah-table-cell-repeated-marker replaces the copied cell content. In this example, “(Continued)” is added to the repeated content.

-ah-repeat-cell-content-at-break is ‘false’ or is unspecified
