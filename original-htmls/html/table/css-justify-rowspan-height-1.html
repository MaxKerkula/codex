Justify the height of table cells

If 'justify-rowspan-height' is true in the Option Setting File, then when a cell spans multiple rows and the height of the cell is higher than the height of the contents of the spanned rows, the height of each spanned row is made as even as possible. 'justify-rowspan-height' is available in AH Formatter V6.6

This feature is specified in the Option Setting File. It applies to all tables in the document

The height of a cell may be increased, depending on the contents of a cell that spans that table row. See example table below.
