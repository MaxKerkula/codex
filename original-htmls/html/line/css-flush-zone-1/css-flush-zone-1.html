Adjusts the space at the end of the last line
Adjusts the space at the end of the last line. If a value (<length> or <percentage>) is specified for '-ah-flush-zone', text-align="justify" is specified but text-align-last="justify" is not specified, and the space at the end of the last line is less than or equal to the specified value, the last line is processed as if text-align-last="justify" is specified. This feature is available in AH Formatter V7.0 or later.
