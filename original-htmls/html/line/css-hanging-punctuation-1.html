Hanging punctuation at the start / end of line

Specifies whether punctuation at the start and/or end of a line may hang outside the line. If -ah-hanging-punctuation: none; is specified, hanging punctuation is not applied. If ‘start’ is specified, hanging punctuation is applied at the start of a line. If text-align: right; or text-align: justify; is specified together with the ‘force-end’ value, then full-width punctuation that comes at the end of a line is forced to hang. The other settings are explained in this sample. See also -ah-hanging-punctuation" in the Online Manual to learn more about the corresponding punctuation.
