Independent line numbering in table columns
Line numbering can be independently applied to a table column by specifying ‘-ah-line-number’ on the table column. All of the line numbering properties that can be used on a page-sequence are available. In this example, line numbers are shown every 5 lines in the left-hand column and every 10 lines in the right-hand column. The line numbers are not being reset on each page.
