Hyphenation processing and SOFT HYPHEN

SOFT HYPHEN (U+00AD) is used to arbitrarily specify the position of the hyphen in the word. If hyphenation does not occur, SOFT HYPHEN does not affect the display result. You can extend ‘hyphenate’ to specify whether to hyphenate by SOFT HYPHEN. The setting of hyphens: true; is available from AH Formatter V7.2.

If hyphenate: true; is specified, normal hyphenation will occur. Hyphenation by dictionary and hyphenation by SOFT HYPHEN are preformed.
