Non-starter Japanese character

Specifies whether to apply the non-starter processing to non-starter Japanese characters under JIS X 4051:2004. The non-starter processing is not applied when ‘-ah-line-break’ is specified as ‘normal’ (a type of mild prohibition). The non-starter processing is applied when specified as ‘strict’ (strict prohibition). As neither ending parenthesis, periods, and mid points are non-starter Japanese characters, they are targets of non-starter processing in either case of ‘normal’ or ‘strict’ setting.
