Barcode Color and Size

Barcode colour can be specified by using the color parameter of the data scheme. With this parameter you can specify any colour other than transparent; however, when the output format is PNG, the specified colour will be converted to RGB. The formatted size of the barcode can be specified with the -ah-content-height and -ah-content-width properties of <img>. Note that changing the aspect ratio or setting a small size may make the barcode impossible to read. Specifying barcode colour is available with AH Formatter V6.3 or later. The Barcode Generator Option must be purchased separately in order to generate barcodes.
