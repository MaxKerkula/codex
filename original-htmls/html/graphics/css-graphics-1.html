Loading external graphics

Itâ€™s possible to load the external graphics by specifying <img src = "PATH/NAME">. The width and the height of the loaded graphics can be specified by content-width and content-height. Antenna House Formatter supports loading the image formats of BMP, JPEG, JPEG2000, PNG, TIFF, GIF, WMF, EMF, EPS, SVG, MathML, CGM and WebP. AH Formatter V7.0 or later supports WebP image format.

No settings to content-width, content-height

-ah-content-width: 5.0cm; -ah-content-height: 3.5cm;
