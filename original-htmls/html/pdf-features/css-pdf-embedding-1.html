PDF Embedding
Antenna House Formatter can use img to embed a PDF like an image. Specify as <img src="embedded.pdf"/>. Fillable PDF forms can also be embedded. A PDF is embedded at its original size. Use '-ah-content-height' and '-ah-content-width' to scale the image if necessary. A page from a PDF file can also be used as a background image, including as the background of an @page rule and page sequence container. For more details, see PDF Embedding in the Manual.
