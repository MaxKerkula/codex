Form Field with Page Number Suffix

The -ah-field-name-suffix-page-number property adds a page number to the name specified by -ah-field-name. This creates unique field names for each page, which is useful when the same form field appears on multiple pages.

This feature is available in AH Formatter V6.6 and later versions.

<input class="demo-field" style="
    -ah-field-name: 'InputText-Page';
    -ah-field-name-suffix-page-number: 1;">
Page 1 Form Field:
Result: The field name on this page becomes "InputText-Page1"
Page 2 Form Field:
Result: The field name on this page becomes "InputText-Page2"

Each page automatically gets a unique field name by appending the page number to the base name specified in -ah-field-name.
