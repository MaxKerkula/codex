Rich Media Annotations
By specifying -ah-multimedia-treatment: richmedia;, you can embed multimedia data in your PDF as "Rich Media" annotations. For example: <video src="video.mp4" type="video/mp4" style="-ah-multimedia-treatment: richmedia;"></video>. If -ah-multimedia-treatment: richmedia-windowed; is specified, the content will be displayed and played in a separate window. Rich media annotations do not require external players for playback. Supported data formats and valid properties differ from when multimedia is embedded using -ah-multimedia-treatment: auto;. For further information on '-ah-multimedia-treatment', please refer to the Online Manual. Rich media annotations are available with AH Formatter V6.5. This feature is supported only with PDF1.7 or later.
-ah-multimedia-treatment="richmedia"
-ah-multimedia-treatment="richmedia-windowed"
-ah-media-skin-control="all", -ah-media-skin-color="red", -ah-media-skin-auto-hide="false"
You can specify -ah-media-skin-control, -ah-media-skin-color, and -ah-media-skin-auto-hide to set the appearance of the skin with the controls to play and stop the rich media.
-ah-media-activation
-ah-media-activation specifies when to enable or disable rich media. For details on allowed values, please refer to '-ah-media-activation' in the Online Manual. The following sample combines PageOpen (activates when the page content containing the content is opened) and PageClose (deactivates when the page containing the content is closed).
-ah-media-activation="PageOpen PageClose"
