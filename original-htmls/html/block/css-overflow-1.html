Overflow extension
If text cannot fit within (i.e., overflows) a fixed-size , the original text can be substituted with a different character string by specifying -ah-overflow: replace;. Also, by specifying -ah-overflow-condense: [font-size | font-stretch | ...], you can condense text, reduce the character size, reduce the line spacing, etc.
