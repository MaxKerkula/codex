Filling Columns

The -ah-column-fill property specifies whether to balance the column height. When -ah-column-fill is balance (default), the columns are balanced as much as possible. When it is auto, columns are not balanced, but content before a spanned column is still balanced.

Unless stated otherwise, these samples have orphans: 1;, widows = 1;, and max-height: 6lh + 6pt;
