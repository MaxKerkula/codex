Specifying the annotation creation and modification date.
You can specify the creation date and modification date of PDF annotations with -ah-annotation-createdate and -ah-annotation-modifydate. See The date format for the character strings that can be specified. This feature is available from AH Formatter V7.1.

-ah-annotation-createdate

You can specify the creation date of the annotation. When omitted, it is considered the same value as -ah-annotation-modifydate.

-ah-annotation-modifydate

You can specify the modification date of the annotation. If omitted, it is considered the value of ‘modifydate’ in -ah-document-info.
